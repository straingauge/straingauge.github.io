Introduction to OpenGL ES 3 on the Raspberry Pi

WARNING: The instructions that follow are for the Raspberry Pi 4. They also may potentially harm your Pi OS intallation. Proceed only at your own risk.

Lesson 1

Before we proceed any further we have to ensure that we install all required dependencies for OpenGL ES programing. On the Pi OS, this will entail doing the following:


sudo apt update
sudo apt install libglfw3-dev libsdl2-dev libglm-dev mesa-utils libglew-dev libsdl2-image-dev
